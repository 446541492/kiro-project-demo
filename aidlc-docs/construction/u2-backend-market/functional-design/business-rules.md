# U2 业务规则 - 后端行情模块

## 1. 榜单类型规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| MKT-001 | 支持的榜单类型 | rise（涨幅榜）、fall（跌幅榜）、volume（成交量榜）、amount（成交额榜）、turnover（换手率榜） |
| MKT-002 | 默认榜单 | 未指定类型时默认返回涨幅榜 |
| MKT-003 | 默认数量 | 每个榜单默认返回 20 条，最大 100 条 |
| MKT-004 | 市场过滤 | 支持按市场过滤：SH（沪市）、SZ（深市）、HK（港股）、US（美股），不指定则返回全部 |

## 2. 搜索规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| SCH-001 | 最小关键词 | 搜索关键词至少 1 个字符 |
| SCH-002 | 最大结果数 | 搜索结果最多返回 20 条 |
| SCH-003 | 搜索类型自动判断 | 纯数字按代码搜索，纯大写字母按拼音搜索，其他按名称搜索 |
| SCH-004 | 大小写不敏感 | 搜索不区分大小写 |
| SCH-005 | 排序规则 | 精确匹配优先，前缀匹配次之，模糊匹配最后 |

## 3. 缓存规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| CACHE-001 | 榜单缓存 | 榜单数据缓存 30 秒 |
| CACHE-002 | 搜索缓存 | 搜索结果缓存 5 分钟 |
| CACHE-003 | 单标的缓存 | 单个标的行情缓存 15 秒 |
| CACHE-004 | 基础信息缓存 | 股票基础信息缓存 24 小时 |
| CACHE-005 | 缓存清理 | 惰性清理 + 每 10 分钟定期清理 |

## 4. Tushare API 规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| API-001 | Token 配置 | Tushare Token 从环境变量 TUSHARE_TOKEN 读取 |
| API-002 | 重试策略 | API 调用失败最多重试 3 次，间隔 1 秒 |
| API-003 | 超时设置 | 单次 API 调用超时 10 秒 |
| API-004 | 空数据处理 | API 返回空数据时返回空列表，不抛异常 |
| API-005 | 频率限制 | 遵守 Tushare 免费版频率限制（每分钟 200 次） |

## 5. 错误处理规则

| 错误场景 | HTTP 状态码 | 错误码 | 错误信息 |
|---------|------------|--------|---------|
| 无效的榜单类型 | 422 | INVALID_RANKING_TYPE | 不支持的榜单类型 |
| 搜索关键词为空 | 422 | EMPTY_KEYWORD | 搜索关键词不能为空 |
| 标的代码不存在 | 404 | SYMBOL_NOT_FOUND | 未找到该标的 |
| Tushare API 错误 | 502 | MARKET_DATA_ERROR | 行情数据获取失败，请稍后重试 |
| Tushare API 超时 | 504 | MARKET_DATA_TIMEOUT | 行情数据请求超时 |

## 6. 数据字段映射

| Tushare 字段 | 应用字段 | 说明 |
|-------------|---------|------|
| ts_code | symbol | 标的代码 |
| name | name | 标的名称 |
| close | current_price | 当前价/收盘价 |
| pct_chg | change_percent | 涨跌幅 |
| change | change_amount | 涨跌额 |
| open | open_price | 开盘价 |
| high | high_price | 最高价 |
| low | low_price | 最低价 |
| pre_close | pre_close | 昨收价 |
| vol | volume | 成交量 |
| amount | amount | 成交额 |
| turnover_rate | turnover_rate | 换手率 |
| pe | pe_ratio | 市盈率 |
| pb | pb_ratio | 市净率 |
