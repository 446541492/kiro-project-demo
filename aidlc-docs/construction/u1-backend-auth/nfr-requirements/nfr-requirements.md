# U1 非功能性需求 - 后端认证模块

## 1. 性能需求

| 需求编号 | 需求描述 | 目标值 |
|---------|---------|--------|
| PERF-001 | 注册 API 响应时间 | < 500ms |
| PERF-002 | 登录 API 响应时间 | < 300ms |
| PERF-003 | Token 刷新响应时间 | < 100ms |
| PERF-004 | 2FA 验证响应时间 | < 200ms |
| PERF-005 | 获取用户信息响应时间 | < 100ms |
| PERF-006 | bcrypt 加密时间 | ~300ms（cost factor=12） |

**说明**:
- bcrypt 的 cost factor=12 会导致密码加密/验证约 300ms，这是安全性和性能的平衡
- 注册和登录的响应时间包含 bcrypt 计算时间

## 2. 安全需求

| 需求编号 | 需求描述 | 实现方式 |
|---------|---------|---------|
| SEC-001 | 密码加密存储 | bcrypt，cost factor=12 |
| SEC-002 | JWT 签名 | HS256 算法，密钥从环境变量读取 |
| SEC-003 | TOTP 密钥加密 | 加密存储在数据库中 |
| SEC-004 | 恢复码加密 | bcrypt 哈希存储 |
| SEC-005 | SQL 注入防护 | SQLAlchemy ORM 参数化查询 |
| SEC-006 | 暴力破解防护 | 登录失败锁定机制（3次验证码，5次锁定） |
| SEC-007 | 敏感信息脱敏 | 日志中不记录密码、Token、TOTP 密钥 |
| SEC-008 | CORS 配置 | 仅允许前端域名访问 |
| SEC-009 | 请求频率限制 | 注册/登录接口限制每 IP 每分钟 10 次 |
| SEC-010 | 密码传输安全 | HTTPS（生产环境），开发环境 HTTP |
