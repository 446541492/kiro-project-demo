# U6 业务规则 - 前端行情与自选

## 1. 行情展示规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| MK-001 | 默认榜单 | 首次加载显示涨幅榜 |
| MK-002 | 自动刷新 | 每 30 秒自动刷新当前榜单 |
| MK-003 | 离开停止 | 离开行情页时停止自动刷新 |
| MK-004 | 涨跌颜色 | 涨为红色、跌为绿色、平为灰色 |
| MK-005 | 数据格式 | 价格保留 2 位小数，涨跌幅带 % 号 |
| MK-006 | 成交量格式 | 大数字使用万/亿单位 |

## 2. 搜索规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| SR-001 | 防抖搜索 | 输入后 300ms 触发搜索 |
| SR-002 | 最少字符 | 至少输入 1 个字符才触发搜索 |
| SR-003 | 清空关闭 | 清空输入框时关闭搜索结果 |
| SR-004 | 结果上限 | 最多显示 20 条搜索结果 |

## 3. 自选管理规则

| 规则编号 | 规则名称 | 规则描述 |
|---------|---------|---------|
| WL-001 | 默认选中 | 进入自选页默认选中第一个组合 |
| WL-002 | 空状态 | 无组合或无标的时显示引导提示 |
| WL-003 | 删除确认 | 删除组合需要二次确认 |
| WL-004 | 默认保护 | 默认组合不显示删除选项 |
| WL-005 | 添加反馈 | 添加标的成功后显示提示 |
| WL-006 | 重复提示 | 添加已存在的标的时显示提示 |
