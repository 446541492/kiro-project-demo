## 用户需求概述
开发一个前端系统，对接现有的后端 API（用户 API、行情 API、自选标的列表 API），实现用户注册登录和个性化自选标的管理功能。

核心功能：
1. 手机/邮箱注册、登录（包含新设备校验、图形验证码、2FA）
2. 展示行情榜单
3. 支持标的搜索
4. 展示用户自选组合，可切换展示各组合下的标的列表
5. 对自选组合进行管理（增、删、改名）
6. 对自选组合标的进行管理（新增、移除）
