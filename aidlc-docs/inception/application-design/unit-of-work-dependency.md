# 工作单元依赖矩阵 - Stocks Assist

## 依赖关系图

```
U1 后端认证模块
 |
 +---> U2 后端行情模块
 |      |
 |      +---> U3 后端自选管理模块
 |      |      |
 +---> U4 前端基础架构
 |      |
 |      +---> U5 前端认证与个人中心
 |
 +---> U6 前端行情与自选
        ^         ^
        |         |
   U2,U3,U4 -----+
```

## 依赖矩阵

| 单元 | 依赖于 | 被依赖于 | 依赖类型 |
|------|--------|---------|---------|
| U1 后端认证模块 | 无 | U2, U3, U4, U5 | - |
| U2 后端行情模块 | U1 | U3, U6 | U1 提供 FastAPI 基础和 JWT 中间件 |
| U3 后端自选管理模块 | U1, U2 | U6 | U1 提供数据库基础，U2 提供 MarketService |
| U4 前端基础架构 | U1 | U5, U6 | U1 提供 API 地址用于客户端配置 |
| U5 前端认证与个人中心 | U1, U4 | 无 | U1 提供认证 API，U4 提供前端基础 |
| U6 前端行情与自选 | U2, U3, U4 | 无 | U2 提供行情 API，U3 提供自选 API，U4 提供前端基础 |

## 推荐执行顺序

```
阶段 1:  U1（后端认证模块）
         |
阶段 2:  U2（后端行情模块）  +  U4（前端基础架构）    [并行]
         |                      |
阶段 3:  U3（后端自选管理）  +  U5（前端认证与个人中心） [并行]
         |                      |
阶段 4:  U6（前端行情与自选）
```

## 接口契约

### U1 -> U2 接口
- FastAPI 应用实例和路由注册机制
- JWT 认证中间件（`get_current_user` 依赖注入）
- 数据库会话管理（`get_db` 依赖注入）

### U1 -> U3 接口
- 数据库基础（SQLAlchemy Base、会话管理）
- JWT 认证中间件
- User 模型（外键关联）

### U1 -> U4 接口
- API 基础地址（http://localhost:8000）
- JWT Token 格式（Bearer Token）
- 认证 API 端点定义

### U1 -> U5 接口
- 认证 API 完整端点（注册/登录/登出/刷新/2FA）
- 请求/响应数据格式（Pydantic Schema）

### U2 -> U3 接口
- MarketService.get_batch_quotes() 方法（获取标的实时行情）

### U2 -> U6 接口
- 行情 API 端点（榜单/搜索/行情查询）
- 请求/响应数据格式

### U3 -> U6 接口
- 自选组合 API 端点（CRUD/排序）
- 自选标的 API 端点（CRUD/排序）
- 请求/响应数据格式

### U4 -> U5, U6 接口
- React Router 路由配置
- Zustand Store 接口
- apiClient 实例
- AppLayout 布局组件
- Ant Design 主题 Token
